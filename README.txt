Solar System Simulator

What the program will do - 

 - Have a solar system with user input amount of planets and info on them
 - have moving gravity simulation bt planets and central Sun
 
 1) first mode is overall system view
   -- can see all planets moving around sun
   -- "explosion" if there is a collision (planet pieces scatter and follow gravity)
   -- if collision, then planets break up into asteroids they can hit other planets
   -- 
 2) have "focus mode" where you can see a planets live stats 
   -- speed
   -- direction
   -- proximal planets
   -- collisions info
